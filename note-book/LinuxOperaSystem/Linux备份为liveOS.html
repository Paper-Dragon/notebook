#еҹәдәҺSystembackвјҜе…·е®һзҺ°ubuntuзі»з»ҹзҡ„й•ңеғҸеҪ’жЎЈгҖҒеӨҚеҲ¶е’Ңе…ӢйҡҶе®үиЈ…#

вјҖгҖҒеҮҶеӨҮвјҜдҪң

  1. еҫ…е…ӢйҡҶзҡ„ubuntuзі»з»ҹдё»жңә
  2. UзӣҳпјҢвүҘ8G
  3. ж–°зҡ„дё»жңәпјҲд»Җд№Ҳж“ҚдҪңзі»з»ҹвҪҶйҷҗеҲ¶пјү

вјҶгҖҒжіЁж„ҸдәӢйЎ№

вҫғдё»зЁӢеәҸдёҚиҰҒе®үиЈ…еңЁзі»з»ҹж №вҪ¬еҪ•вҖң/вҖқдёӢпјҢеҗҰеҲҷеҸҜиғҪеҜјиҮҙзЁӢеәҸе…ӢйҡҶдёҚжҲҗеҠҹпјҢе»әи®®е®үиЈ…еңЁ/optвҪ¬еҪ•дёӢдёүгҖҒжӯҘйӘӨпјҡе®үиЈ…systembackвјҜе…· -> еҲ¶дҪңй•ңеғҸ -> еҲ»еҪ•Uзӣҳ -> UзӣҳеҗҜеҠЁе®үиЈ…

е®үиЈ…systembackвјҜе…·

дёӢиҪҪпјҲвјҜе…·еҢ…е·ІдёӢиҪҪпјүпјҡhttps://nchc.dl.sourceforge.net/project/systemback/1.8/Systemback_Install_Pack_v1.8.402.tar.xz

и§ЈеҺӢпјҡtar -xvf ./Systemback_Install_Pack_v1.8.402.tar.xz

е®үиЈ…пјҡcd Systemback_Install_Pack_v1.8.402/ sudo bash install.sh

#SystembackвҪҖжҢҒзҡ„ubuntuжңҖвҫјеҲ°ubuntu16пјҢдёҚиҝҮе®үиЈ…ubuntu18дёҠд№ҹжІЎй—®йўҳпјҢжҸҗзӨәйҖүжӢ©е®үиЈ…зүҲжң¬ж—¶йҖүжӢ©вҖң 4 в”Җ Ubuntu 16.04.X LTS (Xenial Xerus)вҖқ

жү“еҚ°еҮәпјҡ Systemback installation is successful. иЎЁзӨәе®үиЈ…жҲҗеҠҹгҖӮ

жү“ејҖSystembackвјҜе…·пјҢејҖе§ӢеҲ¶дҪңзі»з»ҹй•ңеғҸпјҢе№¶еҶҷвјҠUзӣҳ

еә”вҪӨдёӯвјјпјҲжҢүwinй”®пјүжҗңзҙўжү“ејҖSystembackгҖӮ

жү“ејҖеҗҺпјҢзҷ»еҪ•в»ҡвҫҜйңҖиҰҒиҫ“вјҠвҪӨжҲ·еҜҶз ҒпјҢиҝӣвјҠдё»з•ҢвҫҜпјҡ

зӣҙжҺҘзӮ№еҮ»вҖңеҲӣе»әLiveзі»з»ҹвҖқпјҢиҝӣвјҠ

иҫ“вјҠй•ңеғҸеҗҚз§°пјҢеӢҫйҖүвҖңеҢ…еҗ«вҪӨжҲ·ж•°жҚ®вҪӮд»¶вҖқпјҢзӮ№еҮ»вҖңеҲӣе»әж–°зҡ„вҖқй•ңеғҸпјҢзӯүеҫ…е®ҢжҲҗгҖӮгҖӮгҖӮ

е®ҢжҲҗеҗҺпјҢдјҡжҸҗзӨәеҶҷвјҠUзӣҳпјҢзӮ№вҖңзЎ®е®ҡеҗҺвҖқиҝ”еӣһдё»в»ҡвҫҜ

жҺҘзқҖйҖүдёӯеҲҡжүҚеҲӣе»әеҘҪзҡ„й•ңеғҸвҪӮд»¶гҖҒйҖүдёӯеҫ…еҶҷвјҠзҡ„UзӣҳпјҲUзӣҳдёәext4ж јејҸпјҢеҰӮжһңжңӘиҜ»еҮәпјҢзӮ№еҲ·ж–°жҢүй’®пјүпјҢзӮ№еҮ»вҖңеҗ‘вҪ¬ж Үи®ҫеӨҮеҶҷвјҠвҖқпјҢејҖе§ӢеҲ»еҪ•UзӣҳгҖӮ

зӯүеҫ…еҶҷвјҠе®ҢжҲҗпјҢвјҖдёӘLiveжЁЎејҸй•ңеғҸзҡ„Ubuntuзі»з»ҹзӣҳеҚіеҲ¶дҪңеҘҪдәҶгҖӮ

пјҒпјҒпјҒзү№еҲ«иҜҙжҳҺпјҒпјҒпјҒ

иҜҘжЁЎејҸеҲӣе»әзҡ„й•ңеғҸдёҚжҳҜghostзҡ„ISOвҪӮд»¶пјҢжҳҜвјҖдёӘLiveжЁЎејҸзҡ„й•ңеғҸпјҢеҰӮжһңиҰҒеҲ¶дҪңISOвҪӮд»¶пјҢеҲҷйҖҡиҝҮвҖңиҪ¬еӯҳдёәе…үзӣҳй•ңеғҸвҖқжҢүй’®еҲ¶дҪңгҖӮ

иҝҷвҫҘйҖүжӢ©LiveжЁЎејҸй•ңеғҸеҺҹеӣ пјҲвҖңиҪ¬еӯҳдёәе…үзӣҳй•ңеғҸвҖқжҢүй’®дёҚеҸҜвҪӨпјүжҳҜISOй•ңеғҸвҪӮд»¶дёҚиғҪвјӨдәҺ4GпјҢвҪҪжҲ‘д»¬иҰҒе…ӢйҡҶзҡ„зі»з»ҹеҫҲеҸҜиғҪдјҡи¶…иҝҮ4GпјҢе…ідәҺеҰӮдҪ•еҲ¶дҪң и¶…еҮә4Gзҡ„ISOвҪӮд»¶пјҢд№ҹжңүи§ЈеҶіеҠһжі•пјҢжңүзӮ№еӨҚжқӮпјҢиҝҷвҫҘжңӘеҒҡе°қиҜ•пјҢеҸҜд»ҘеҸӮиҖғпјҡи§ЈеҶіsystemback вҪҶжі•вҪЈжҲҗи¶…иҝҮ4Gзҡ„isoзҡ„й—®йўҳгҖӮ

Uзӣҳе®үиЈ…

UзӣҳжҸ’вјҠж–°зҡ„дё»жңәпјҢbiosи®ҫзҪ®UзӣҳеҗҜеҠЁеҗҺпјҢејҖжңәпјҢ

иҝӣвјҠSystembackзҡ„LiveжҒўеӨҚжЁЎејҸеҗҜеҠЁв»ҡвҫҜпјҢеҮәзҺ°зҡ„з”»вҫҜеҸҜиғҪжҳҜд№ұз ҒпјҢдёҚвҪӨз®ЎпјҢзӣҙжҺҘеӣһв»ӢпјҢиҝӣвјҠLiveжЁЎејҸзҡ„ubuntuзі»з»ҹ

еә”вҪӨзЁӢеәҸвҫҘжүҫеҲ°Systembackеә”вҪӨпјҢжү“ејҖпјҢйҖүжӢ©вҖңзі»з»ҹе®үиЈ…вҖқ

иҫ“вјҠж–°зі»з»ҹзҡ„вҪӨжҲ·еҗҚгҖҒеҜҶз ҒгҖҒдё»жңәеҗҚпјҢдёӢвјҖжӯҘ

иҝӣвјҠзі»з»ҹе®үиЈ…в»ҡвҫҜпјҢиҝҷвҫҘиҰҒжіЁж„ҸиҰҒеҒҡзі»з»ҹеҲҶеҢәпјҢдёӢвҫҜз»ҶиҜҙ

дёҠвҫҜдёӨдёӘдё»еҲҶеҢәпјҢвјҖдёӘжҳҜдё»жңәзЎ¬зӣҳпјҢвјҖдёӘжҳҜUзӣҳпјҲзңӢеҲҶеҢәвјӨвј©пјүпјҢвҪ¬ж ҮжҳҜе®үиЈ…еңЁдё»жңәзЎ¬зӣҳдёҠгҖӮ

йңҖиҰҒеҲҶй…Қ4дёӘеҲҶеҢә

вјҖиҲ¬и®ҫзҪ®4дёӘеҢәй—ҙеҚіеҸҜпјҢ"/"(ext4)пјҢ вҖң/homeвҖқ(ext4), вҖңswapвҖқ, вҖңboot/efiвҖқгҖӮе…¶дёӯ

  1. е…ҲеҲҶй…Қ"swap"пјҢвјӨвј©дёҺеҶ…еӯҳзӣёеҪ“еҚіеҸҜпјҢжҲ‘и®ҫзҪ®зҡ„жҳҜ4G
  2. еҶҚеҲҶй…Қ"/bootвҖқпјҢвјӨвј©200mеҚіеҸҜпјҢжҲ‘и®ҫзҪ®зҡ„жҳҜ500M
  3. еҶҚеҲҶй…ҚвҖң/вҖқпјҢвҪӮд»¶зі»з»ҹйҖүext4пјҢеӣ дёәжҲ‘д»¬зҡ„еә”вҪӨзЁӢеәҸе’Ңж•°жҚ®йғҪеңЁзі»з»ҹж №вҪ¬еҪ•вҖң/вҖқжҲ–иҖ…/вҖңoptвҖқвҪ¬еҪ•дёӢпјҢжүҖд»ҘиҝҷвҫҘиҰҒеҲҶй…ҚжӣҙеӨҡз©әй—ҙпјҢвҫ„е°‘30G
  4. жңҖеҗҺзҡ„з©әй—ҙеҲҶй…Қз»ҷвҖң/homeвҖқпјҢвҪӮд»¶зі»з»ҹйҖүext4пјҢйў„з•ҷвҫңеӨҹз©әй—ҙеӯҳж”ҫзі»з»ҹвҪӨжҲ·ж•°жҚ®пјҲ/жЎҢвҫҜ/вҪӮжЎЈ/дёӢиҪҪ/еӣҫвҪҡзӯүиө„ж–ҷпјү

иҰҒйҮҚж–°еҲҶеҢәеүҚе…ҲеҲ йҷӨе·ІжңүеҲҶеҢә

еҲҶеҢәжҢӮиҪҪеҘҪд№ӢеҗҺпјҢжңҖеҗҺеӢҫйҖүвҖңдј йҖ’вҪӨжҲ·й…ҚвҪӮд»¶еҸҠвҪӨжҲ·ж•°жҚ®вҖқпјҢзӮ№еҮ»вҖңдёӢвјҖжӯҘвҖқпјҢејҖе§Ӣе®үиЈ…зі»з»ҹ
