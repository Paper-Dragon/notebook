安装Nvidia驱动

PaperDragon2023年8月13日...小于 1 分钟

安装Nvidia驱动

安装驱动依赖

apt install gcc g++ make -y

# 删除原有开源驱动包
apt-get remove --purge nvidia*

安装驱动

apt install nvidia-driver-470 -y  # 其中470 为版本号

# 重启来激活启动
reboot

检查是否安装成功

nvidia-smi
