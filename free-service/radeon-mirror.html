Radeon镜像站

Amd的Radeon镜像站

最近在使用 Ubuntu 安装Amd显卡驱动的时候访问国外镜像站网速特别慢，简直无法忍受，于是乎我自己做了个镜像站，用于加速驱动的下载。

下面的方法仅适用于Ubuntu，其他操作系统按照链接内容安装并替换镜像站 我没用到，没写，谁用到了写一个 https://rocm.docs.amd.com/projects/install-on-linux/en/latest/install/amdgpu-install.html

镜像内容

镜像的内容

一键切换 amdgpu-install安装器 镜像站脚本

curl -sSL https://www.geekery.cn/sh/radeon/set_radeon_mirror.sh | bash

安装教程

打开网站从下面的目录中找到你的显卡

AMD驱动下载地址: https://www.amd.com/zh-cn/support/download/drivers.html

进去之后找到你想安装的版本,这里是闭源驱动。

使用如下命令下载安装器并设置镜像源，这时设置的镜像源是国外的镜像源。

sudo apt install amdgpu-install_[版本号]_all.deb

这个版本号根据自己下载的文件进行更改，直接Tab就行。

切换镜像站

方法一: 手动切换镜像站

打开 /etc/apt/source.list.d/

root@b550m:/etc/apt/sources.list.d# cat rocm.list
deb [arch=amd64] https://repo.radeon.com/rocm/apt/[版本号] [版本号] main
root@b550m:/etc/apt/sources.list.d# cat amdgpu.list
deb https://repo.radeon.com/amdgpu/[版本号]/ubuntu [系统版本号] main
#deb-src https://repo.radeon.com/amdgpu/[版本号]/ubuntu [系统版本号] main

编辑里面的域名并替换为镜像的域名

rocm.list

deb [arch=amd64] https://radeon.geekery.cn/rocm/apt/[版本号] [系统版本号] main

amdgpu.list

deb https://radeon.geekery.cn/amdgpu/[版本号]/ubuntu [系统版本号] main
#deb-src https://repo.radeon.com/amdgpu/[版本号]/ubuntu [系统版本号] main

方法二：使用我写的一键切换脚本

curl -sSL https://www.geekery.cn/sh/radeon/set_radeon_mirror.sh | bash

执行安装

amdgpu-install
