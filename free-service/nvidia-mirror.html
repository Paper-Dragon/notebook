Nvidia镜像站

在 Ubuntu 系统中使用 Nvidia 显卡时，下载驱动速度可能会非常慢。为了解决这个问题，我搭建了一个镜像站来加速下载。

镜像站提供两个主要地址：

开发者工具镜像站

  • 地址：nvidia-developer.geekery.cn
  • 包含 CUDA 工具等 Nvidia 开发必需资源。

Docker Runtime 镜像站

  • 地址：nvidia-docker.geekery.cn
  • 提供 libnvidia-container，用于支持容器内的 Nvidia 显卡运行。

  • 预览libnvidia-container 镜像源 https://nvidia-docker.geekery.cn/libnvidia-container/stable/

  • 预览cuda 镜像仓库 https://nvidia-developer.geekery.cn/compute/cuda/repos/

一键部署Nvidia-Docker脚本

curl -sSL https://www.geekery.cn/sh/LinuxCTS/os/apt/nvidia-docker.sh | bash

脚本中一键部署了

  • cuda工具
  • docker runtime
  • docker

支持系统版本

操作系统版本备注
Ubuntu18.04, 20.04, 22.04-
Debian10, 11-
WindowsWSL - Ubuntu需在Windows内手动安装 Nvidia 驱动

一键部署Nvidia驱动

可以通过以下脚本快速安装 Nvidia 驱动：

curl -sSL https://www.geekery.cn/sh/LinuxCTS/os/apt/nvidia-driver.sh | bash
